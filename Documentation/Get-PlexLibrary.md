# Get-PlexLibrary

## SYNOPSIS
By default, returns a list of libraries on a Plex server.

## SYNTAX

```
Get-PlexLibrary [[-Id] <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
By default, returns a list of libraries on a Plex server.
